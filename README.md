# 🚀 ChurnIQ: Predictive E-Commerce Retention Engine

## 📊 Overview
ChurnIQ (formerly PrimePawz Analytics) is a Machine Learning web application designed to help e-commerce managers predict customer churn before it happens. By analyzing historical purchase data, this AI-driven tool identifies high-risk customers, allowing businesses to proactively deploy targeted marketing campaigns and maximize customer lifetime value.

### 🔗 Live Application
**[Click here to view the live interactive dashboard!](https://primepawz-ai-nz6j6eqzvqcq5vzm39jmwz.streamlit.app/)**

## ✨ Key Features
* **Predictive Machine Learning:** Utilizes a Scikit-Learn Decision Tree Classifier to predict customer behavior based on visit frequency and monetary value.
* **Instant Business Value:** Translates raw data into actionable business recommendations (e.g., "Send VIP Email" vs. "Send 20% Win-Back Discount").
* **Interactive UI:** Features a user-friendly, responsive frontend built entirely in Python using Streamlit, requiring no technical expertise from the end-user.

## 🛠️ Technology Stack
* **Language:** Python
* **Data Processing:** Pandas
* **Machine Learning:** Scikit-Learn
* **Web Framework:** Streamlit
* **Deployment:** Streamlit Community Cloud & GitHub
